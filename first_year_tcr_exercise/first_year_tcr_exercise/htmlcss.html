Wat is HTML en CSS ?

HTML en CSS worden namelijk gebruikt om websites te bouwen en vorm te geven. Waar je met HTML de inhoud en de indeling van een webpagina kan bepalen, wordt CSS gebruikt om de webpagina van vormgeving te voorzien.
